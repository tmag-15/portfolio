05 / STEALTH

drone platform — in development · first milestone shipped, sim-verified

Drone
Autonomy

A safety-critical platform for autonomous drone fleets — built simulation-first, so the entire system runs end to end before a single propeller spins.

TypeScript services and Python drone agents meet over MQTT, and the simulator speaks the exact message contracts real hardware will — the server never knows it's flying a fake. A deterministic risk engine fuses eight weighted sensor signals into a 0–100 score and a bounded set of actions; a ten-state lifecycle FSM and on-drone failsafes handle disconnects, geofence breaches, and low battery with no server round-trip; every command is written to an append-only event store before it executes. Milestone one — a five-minute automated scenario running end to end with zero errors — is green, and a 41-finding code audit has been driven to zero.

06

golf-sim — CV launch monitor + UE5 renderer · in development

Golf
Simulator

A camera-based golf launch monitor built from scratch — high-speed stereo vision measures the shot, an Unreal Engine 5 world flies it. Both ends of the pipe, one wire contract.

The Python/OpenCV backend runs 2–3 high-speed cameras (120–1000 fps) through a ten-phase pipeline — detection, 2D Kalman tracking, chessboard calibration, sub-frame clock sync, stereo triangulation, 3D tracking, launch-event extraction, least-squares spin estimation — chasing TrackMan-class accuracy. Each shot leaves as versioned JSON over WebSocket into an Unreal Engine 5 frontend that computes ball flight analytically at 120 Hz — gravity, drag, Magnus, deterministic and automation-tested, no engine physics — and renders it under Lumen. Two full audit passes closed; all twelve test modules green.
